# TRPG System Vault

This vault serves as the comprehensive documentation and design space for a custom online tabletop role-playing game system, inspired by D&D and similar systems.

## System Components

This TRPG system includes:

### Core Systems
- **[[Stats System]]** - Primary/secondary stats, scaling dice rolls, pseudo-infinite progression
- **[[Ability Tree]]** - Classless progression, unlockable abilities with stat prerequisites
- **[[Character Creation]]** - Classless character building, races, backgrounds

### Gameplay Systems
- **[[Spells]]** - Magic system, spell lists, casting mechanics
- **[[Combat]]** - Battle mechanics overview
- **[[Combat System]]** - Detailed resource-based combat rules (HP, Stamina, Mana)
- **[[Basic Actions and Conditions]]** - Universal actions (jump, hide, grapple, etc.) and status effects
- **[[Equipment]]** - Weapons, armor, items, and gear
- **[[Social Interactions]]** - Dialogue systems, persuasion, relationships
- **[[Game Mechanics]]** - Core rules, dice systems, progression

Use this vault to document rules, create content, and develop the game system. Link related concepts together to build a interconnected knowledge base for your TRPG.
