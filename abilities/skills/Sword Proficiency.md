# Sword Proficiency

Basic competence with all sword types.

## Type
Weapon Proficiency - Passive

## Prerequisites
- **Category:** [[Swords]]
- **Stats Required:** Might 4 OR Agility 4

## Effect
You can wield swords effectively without penalty.

- No disadvantage on sword attacks
- Use proper damage dice for swords
- Can use sword special properties

## Weapons Unlocked
- Shortsword (1d6, light)
- Longsword (1d8, versatile)
- Rapier (1d8, finesse)
- Scimitar (1d6, finesse, light)
- Greatsword (2d6, heavy, two-handed)

## Scaling
None - this is a basic proficiency.
