# Polearm Proficiency

Basic competence with reach weapons.

## Type
Weapon Proficiency - Passive

## Prerequisites
- **Category:** [[Polearms and Halberds]]
- **Stats Required:** Might 6

## Effect
You can wield polearms and halberds effectively without penalty.

- No disadvantage on polearm attacks
- Use proper damage dice
- Can use reach property (10 feet)

## Weapons Unlocked
- Spear (1d6, thrown, versatile, reach)
- Pike (1d10, heavy, reach, two-handed)
- Glaive (1d10, heavy, reach, two-handed)
- Halberd (1d10, heavy, reach, two-handed)
- Quarterstaff (1d6, versatile)

## Scaling
None - this is a basic proficiency.
